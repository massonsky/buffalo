## ✅ Реализовано

### 1. CLI Interface (Cobra)
**4 команды:**
- `buffalo build` - Сборка proto файлов
- `buffalo init` - Инициализация проекта
- `buffalo version` - Информация о версии
- `buffalo completion` - Shell completions

**Флаги:**
- `--config` - Путь к конфигу
- `--verbose` - Debug логирование
- Per-command флаги (--lang, --output, --dry-run и др.)

### 2. Configuration System (Viper)
- YAML конфигурация с валидацией
- Environment variables (BUFFALO_*)
- Разделы: project, proto, output, languages, build, logging
- 18 тестов, 93.6% coverage

### 3. Project Initialization
- `buffalo init` создаёт полную структуру
- buffalo.yaml с комментариями
- Директории (./protos, ./generated)
- Example proto файл
