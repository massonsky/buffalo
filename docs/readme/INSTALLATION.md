# 📦 Installation

## Quick install

### Linux/macOS

```bash
curl -sSL https://raw.githubusercontent.com/massonsky/buffalo/main/scripts/install.sh | bash
```

### Windows (PowerShell)

```powershell
irm https://raw.githubusercontent.com/massonsky/buffalo/main/scripts/install.ps1 | iex
```

### Verify

```bash
buffalo --version
```

## Alternative methods

### Go install

```bash
go install github.com/massonsky/buffalo/cmd/buffalo@latest
```

### Build from source

```bash
git clone https://github.com/massonsky/buffalo.git
cd buffalo
make install
```
