# Buffalo - Краткое резюме проекта

## 📋 Общая информация

**Название:** Buffalo  
**Тип:** Кроссплатформенный мультиязычный сборщик protobuf/gRPC файлов  
**Язык разработки:** Go 1.21+  
**Лицензия:** MIT  
**Статус:** В разработке (Pre-release)

---

## 🎯 Цель проекта

Создать универсальный, производительный и простой в использовании инструмент для компиляции protobuf/gRPC файлов с поддержкой множества языков программирования.

### Проблемы, которые решает Buffalo:

1. **Сложность настройки компиляции proto файлов** для разных языков
2. **Отсутствие единого инструмента** для мультиязычных проектов
3. **Медленная компиляция** больших proto файлов
4. **Сложность интеграции** в CI/CD
5. **Отсутствие инкрементальной сборки** в стандартных инструментах

---

## 🛠️ Технологический стек

### Основные:
- **Go 1.21+** - основной язык
- **Cobra** - CLI framework
- **Viper** - конфигурация
- **protoc** - protobuf компилятор

### Инструменты разработки:
- **golangci-lint** - линтинг
- **testify** - тестирование
- **GitHub Actions** - CI/CD
- **Docker** - контейнеризация

### Поддерживаемые компиляторы:
- **Python:** protoc + grpcio-tools
- **Go:** protoc-gen-go + protoc-gen-go-grpc
- **Rust:** prost + tonic
- **C++:** protoc + grpc_cpp_plugin
